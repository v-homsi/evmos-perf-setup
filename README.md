# evmos-perf


### Run

```bash
docker-compose up
```

To stop:

```bash
docker-compose down --volumes
```

### Customize

You can update the commits to be tested by updating `commit_hash` for `evmos-devnet1` and `evmos-devnet2` in `docker-compose.yml`.
